Mine Sweeper Wix

Instruction
  1. Click on cell reveals the value underneath it:
    1. If it is a mine, you lose.
    2. Otherwise, display the number of mines around the cell (or empty if there are no mines around)
    3. If there are no mines around the cell, reveal all cells around it and and all cells around any adjacent empty cell.
  2. Shift button + Left Mouse Click puts or removes a flag on that cell. (and updates the number of remaining flags)
  3. A flagged cell cannot be revealed (click does nothing) until the flag is removed.
If all mines are flagged correctly, you win.
